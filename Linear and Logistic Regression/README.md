# 🚀 Regression and Classification From Scratch

This repository contains implementations of core **Machine Learning algorithms** built from scratch using **NumPy** and **Matplotlib**.

The goal is to understand how these algorithms work internally without using libraries like **scikit-learn**.

---

## 📁 Project Structure

| File Name                                   | Description                                     |
| ------------------------------------------- | ----------------------------------------------- |
| `linear_regression.ipynb`                   | Linear Regression using Gradient Descent        |
| `polynomial_regression.ipynb`               | Polynomial Regression for non-linear data       |
| `logistic_regression.ipynb`                 | Logistic Regression for binary classification   |
| `logistic_regression_with_polynomial.ipynb` | Logistic Regression with polynomial features    |
| `Regularized_logistic_regression.ipynb`     | Logistic Regression with L2 Regularization      |
| `vectorization.ipynb`                       | Efficient computation using NumPy vectorization |
| `polynomial_data.csv`                       | Dataset for polynomial regression               |

---

## 🔧 Features

* Built completely from scratch using **Python & NumPy**
* No use of ML libraries like **scikit-learn**
* Gradient Descent implementation
* Regularization to prevent overfitting
* Vectorized computations for efficiency
* Data visualization using **Matplotlib**

---

## 📦 Requirements

* Python 3.x
* NumPy
* Matplotlib *(optional for plots)*

Install dependencies using pip:

```bash
pip install numpy matplotlib
```
